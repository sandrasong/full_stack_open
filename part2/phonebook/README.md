# Phonebook

This SPA is a phonebook. Users are able to add a person, delete a person, and update a person's number. A search input as the filter is implemented. The data is stored in db.json.

## Start the SPA
First install packages
```bash
# Install packages
$ npm install
```

Start the local server in a console windown
```bash
# Start JSON server, same as npx json-server --p 3001 db.json
$ npm run server
```

Start the frontend
```bash
$ npm run dev
```

Server runs at http://localhost:3001/, frontend runs at http://localhost:5173/.
