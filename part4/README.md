# BlogList backend

This is a blog list application, that allows users to save information about interesting blogs they have stumbled across on the internet. For each listed blog we will save the author, title, URL, and amount of upvotes from users of the application.

## Start the local application
First install packages
```bash
# Install packages
$ npm install
```

Then start the application in terminal

`$ npm start` or `$ npm run dev`

Server runs at http://localhost:3003/
