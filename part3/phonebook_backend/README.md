# Phonebook backend

This is the backend application of the phonebook project. The tech stack is Node.js with Express. 

## Start the local application
First install packages
```bash
# Install packages
$ npm install
```

Then start the application in terminal

`$ npm start` or `$ npm run dev`

Server runs at http://localhost:3001/
