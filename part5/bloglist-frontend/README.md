# BlogList frontend

This is a blog list application, that allows users to save information about interesting blogs they have stumbled across on the internet. For each listed blog we will save the author, title, URL, and amount of upvotes from users of the application.

## Start the SPA
First install packages
```bash
# Install packages
$ npm install
```

Start the local server in a console windown
```bash
# Start JSON server, same as npx json-server --p 3001 db.json
$ npm run server
```

Start the frontend
```bash
$ npm run dev
```

Server runs at http://localhost:3003/, frontend runs at http://localhost:5173/.
